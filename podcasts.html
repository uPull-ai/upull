A shareable HTML version of the prompt pack for collaborators working on NotebookLM episode drafts.

Shared Notebook Setup

  1. Upload the three core source files into the notebook: community.html, academy.html, and AI_Healthcare_Agents_Case_Studies_v6.docx.
  2. Paste the episode prompt under a short one-line brief.
  3. Generate Audio Overview in NotebookLM and review the first draft.
  4. Tighten the prompt if the output drifts away from the source material.

Episode 01

Why AI productivity now

Source focus: community + academy + case-study overview.

NotebookLM prompt: Create a grounded, practical podcast episode about why healthcare AI productivity matters now. Focus on workflow bottlenecks, clinician time recovery, and the idea of starting with one measurable use case.

Host script outline:

  • Open with the practical problem or opportunity.
  • Use one clear example from the shared source bundle.
  • Explain the workflow, adoption, or governance lesson.
  • Close with one action step for the audience.
